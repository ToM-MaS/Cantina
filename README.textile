== Welcome to Cantina

Cantina is a provisioning system for SIP phones.

== Howto install Cantina on a Debian 5 system

Use a fresh installed Debian 5 system. Login as root and copy and paste some stuff we need to install in advance:

aptitude -y install curl git-core patch file

aptitude -y install build-essential bison  \
  openssl zlib1g-dev libssl-dev libreadline5-dev libxml2-dev \
  libreadline5-dev libxml2-dev

echo \
  'deb http://ftp.de.debian.org/backports.org/ lenny-backports main' \
  > /etc/apt/sources.list.d/lenny-backports.list

aptitude update

aptitude -y -t lenny-backports install \
  sqlite3 libsqlite3-dev

Now you want to add a new account to the system:

adduser cantina

and su to that account:

su - cantina

Now we install RVM for our Ruby on Rails environment:

bash < <( curl http://rvm.beginrescueend.com/releases/rvm-install-head )

cp .bashrc .bashrc-ORIG

echo '[[ -s $HOME/.rvm/scripts/rvm ]] && source $HOME/.rvm/scripts/rvm' >> .bashrc

perl -pi -e 's/\[ -z \"\$PS1\" \] \&\& return/if \[\[ -n \"\$PS1\" \]\] \; then/' .bashrc

echo 'fi' >> .bashrc

Now you have to logout and login again as the use cantina

exit

su - cantina

After that we use RVM to install Ruby 1.9.2:

rvm install ruby-1.9.2

rvm --default 1.9.2

Now install Ruby on Rails 3 with:

gem install rails

Now it's time to get the Cantina software:

git clone git://github.com/amooma/Cantina.git

Just 5 more steps:

cd Cantina

bundle install

rake db:migrate

rake db:seed

rails server --binding=your.ip.add.ress
